# 區段比對預先分享見解

在您決定與策略合作夥伴共用資料時，了解客戶配對情況非常重要，這樣您就知道資料共用將有多大幫助。「區段比對」可讓您在共用任何資料之前查看與潛在資料合作夥伴的重疊。
