# 建立區段

了解如何在Adobe Experience Platform中建立區段。 建立區段規則，並將它們套用至您的即時客戶設定檔，以產生可支援客戶體驗的受眾。 如需詳細的產品檔案，請參閱 [在UI中建立區段](https://experienceleague.adobe.com/docs/experience-platform/segmentation/ui/overview.html) 和 [使用API建立區段](https://experienceleague.adobe.com/docs/experience-platform/segmentation/tutorials/create-a-segment.html).

如需詳細資訊，請參閱 [區段產生器檔案](https://experienceleague.adobe.com/docs/experience-platform/segmentation/ui/segment-builder.html).
