# 建立資料使用原則

瞭解如何在Adobe Experience Platform使用者介面中建立和管理資料使用原則。 資料使用原則可讓您根據資料標籤，限制特定目的的資料使用。若要瞭解API型方法，請參閱[使用API建立資料使用原則](https://experienceleague.adobe.com/docs/experience-platform/data-governance/policies/create.html)。
