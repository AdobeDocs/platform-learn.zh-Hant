# 區段比對資料治理

瞭解如何在Real-Time CDP中設定和使用資料治理控制，以便限制可與資料合作夥伴共用的資料集（以及使用這些資料集的區段）。 如需詳細資訊，請參閱[檔案](https://experienceleague.adobe.com/zh-hant/docs/experience-platform/segmentation/ui/segment-match/overview){target="_blank"}。

>[!IMPORTANT]
>
>Adobe於2021年推出區段比對，供客戶共同作業和交換受眾。 2025年初，Adobe推出Real-Time CDP Collaboration，這是符合此使用案例的長期方法。
>
> * 美國、加拿大、澳洲和紐西蘭的客戶： Adobe建議Real-Time CDP Prime和Ultimate客戶將資料共同作業使用案例從「區段比對」轉移至Real-Time CDP Collaboration。 檢視Real-Time CDP Collaboration的[檔案](https://experienceleague.adobe.com/zh-hant/docs/real-time-cdp-collaboration/using/home){target="_blank"}和[快速入門手冊](https://experienceleague.adobe.com/zh-hant/docs/real-time-cdp-collaboration/using/quick-start-guide){target="_blank"}，並連絡您的Adobe客戶團隊以深入瞭解。
> * 對於所有其他地區的客戶：在2026年在這些地區發行Real-Time CDP Collaboration之前，建議使用「區段比對」選項。
