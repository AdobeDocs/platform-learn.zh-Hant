# 刪除設定檔

瞭解如何使用即時客戶設定檔API從設定檔存放區中刪除資料。 使用設定檔API，您可以從設定檔存放區中移除資料，而不會影響資料湖或身分圖表。 疑難排解身分圖表問題，以及修正只影響少數設定檔的資料擷取中偶爾發生的錯誤時，這項功能會很有用。
