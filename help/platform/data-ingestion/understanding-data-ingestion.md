# 資料擷取概觀

Adobe Experience Platform的資料擷取功能可讓您將資料一起整合到一個開放且可擴充的平台中，以便管理即時客戶個人檔案。 如需詳細資訊，請瀏覽[資料擷取檔案](https://experienceleague.adobe.com/docs/experience-platform/ingestion/home.html?lang=zh-Hant)。
