# 監視資料內嵌

瞭解如何使用監視儀表板來監視和追蹤內嵌到 Adobe Experience Platform 中的資料。此監控儀表板提供透過Data Lake到Profile和Identity Services的來源資料處理自上而下的檢視，並包含可操作的深入分析。
