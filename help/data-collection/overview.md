# 資料收集教學課程

Data Collection是新一代Adobe的網站標籤和行動SDK管理功能。 資料收集可讓客戶透過簡單的方式部署及管理必要的分析、行銷及廣告解決方案，以便支援相關客戶體驗。 使用這些影片和教學課程來協助學習此關鍵技術。
