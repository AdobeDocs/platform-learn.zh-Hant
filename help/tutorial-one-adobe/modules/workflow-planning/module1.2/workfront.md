# 1.2使用Adobe Workfront工作流程管理工作流程

## 學習目標

完成本實驗後，您將熟悉下列專案：

- 瞭解Workfront物件的階層，包括投資組合、方案、專案和任務
- 使用為行銷活動量身打造的範本，建立新專案。
- 依角色和可用性將任務指派給資源。
- 上傳檔案並請求核准
- 將核准的檔案傳送到AEM Assets CS和內建整合

## 先決條件

- 使用Workfront standard/planner授權存取Adobe Workfront。
