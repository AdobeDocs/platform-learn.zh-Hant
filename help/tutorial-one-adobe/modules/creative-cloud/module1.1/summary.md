# Firefly服務的摘要和優點

Firefly服務是一組完整的創作AI和創作API，可自動化工作流程。 透過將這些功能內嵌到任何生產或體驗工作流程中，組織可以擴展內容產生、編輯和組裝，自動化內容的生產，同時維持品質和控制。
