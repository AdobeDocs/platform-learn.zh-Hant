# 1.8使用Workfront、Frame.io和企業儲存管理進行統一的檢閱和核准

Workfront + Frame.io整合將創意操作從一系列中斷連線的步驟轉換為單一連線體驗。 如此一來，組織就能在更短的時間內製作更多內容、更高品質，同時讓每個利害關係人獲得所需的曝光度和信心。

透過將Workfront的專案和工作管理功能直接與Frame.io的即時稽核和共同作業體驗連線，團隊可消除移交、重複更新和狀態追蹤。

Creative從規劃到執行和審查的工作流程非常順暢，使團隊能夠：

- 從簡短的資產移至延遲較少的最終資產
- 保持意見反應與情境相關，集中處理，縮短稽核週期
- 保持專案移動而不會切換工具或失去動力

## 先決條件

- 存取部署在同一IMS組織中的Workfront和Frame.io。
- 必須在Adobe Unified Experience和Adobe企業儲存空間上啟用Workfront執行個體。
