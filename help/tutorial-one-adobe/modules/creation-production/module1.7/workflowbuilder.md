# 1.7 Firefly Creative Production for Enterprise

Firefly Creative Production for Enterprise的主要焦點在於建立和設定工作流程的創意技術人員和解決方案建置者。

透過Firefly Creative Production for Enterprise，您可以：

- 組建工作流程：使用視覺化介面及手動或AI協助建立多步驟生產工作流程。
- 執行工作流程：透過在瀏覽器外部運作的強大執行階段，大規模執行工作流程，包括批次處理。
- 部署工作流程：將工作流程發佈為API端點或程式化UI，以便整合至其他工具和系統。
- 整合工作流程：將工作流程連結至DAM、啟用串流和內容供應鏈等企業系統。

支援的主要使用案例包括以下內容：

- 社群媒體頻道調整，例如為每個社群媒體平台和型別建立大小變體。
- 資產準備，例如準備資產以用於其他表面，例如列印或網頁。
- 內容重新目標定位，例如重新整理及重複使用視覺效果，以重新吸引感興趣的對象。
- 電子商務，例如標準化產品影像以及為不同受眾建立區段專用變體。
