# Analytics與代理程式

## 學習目標

在本單元中，您將使用外部非Adobe Analytics代理程式來分析Customer Journey Analytics和/或Adobe中的資料。

## 先決條件

- 存取Adobe Analytics (AA)和/或Adobe Customer Journey Analytics (CJA)
- 透過付費訂閱存取ChatGPT和/或Claude.ai

## 練習

[1.5.1 CJA與ChatGPT搭配MCP伺服器](./ex1.md){target="_blank"}

在本練習中，您將會在ChatGPT中設定自訂MCP伺服器，以便將ChatGPT直接連線至Customer Journey Analytics，而且您將直接在ChatGPT中開始分析在Adobe Customer Journey Analytics (CJA)中託管的資料。

[1.5.2 CJA與Claude.ai搭配MCP伺服器](./ex2.md){target="_blank"}

在本練習中，您將會在Claude.ai中設定自訂MCP伺服器，以直接將Claude.ai連線至Customer Journey Analytics，並且您將直接從Claude.ai開始分析Adobe Customer Journey Analytics (CJA)中託管的資料。

[1.5.3 Adobe Analytics與ChatGPT搭配MCP伺服器](./ex3.md){target="_blank"}

在本練習中，您將會在ChatGPT中設定自訂MCP伺服器，以便將ChatGPT直接連線至Adobe Analytics，而且您將開始直接從ChatGPT分析Adobe Analytics (AA)中託管的資料。
