# 快速入門 — GenStudio

在本單元中，您將設定所有專案，讓您的環境和電腦準備好開始使用GenStudio Tech Labs。

## 學習目標

- 瞭解如何設定Chrome擴充功能
- 設定您的Adobe I/O專案和Postman或Postbuster
- 安裝參加GenStudio技術實驗室所需的應用程式

在本練習中，您將設定您的Adobe I/O專案，以用於與各種Adobe API端點互動。
