# 深入分析：概觀

讓我們看看能為您提供行銷活動深入分析的大量報表。  這些只是範例。  還有許多其他現成可用的功能，建置或修改報表相當容易。

您需要記住的一件事是，平台上的任何欄位（原生或由您建立的欄位，即自訂欄位）都可以成為報表的一部分。  因此，選擇幾乎是無限的。
